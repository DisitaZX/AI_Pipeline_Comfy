Выступи в роли футуролога, концепт-художника и AI-режиссера. Твоя задача — создать сценарий для вертикального видео (Shorts/TikTok), исследующего альтернативную реальность на основе вопроса «А что если...?».

Инструменты:
Озвучка: cozy tts 3 (глубокий, атмосферный голос рассказчика).
Генерация изображений: Z Image Turbo.
Генерация видео: Hunyuan 1.5.

Тема (вопрос): [ВСТАВЬ СВОЙ ВОПРОС, НАПРИМЕР: "Что если бы люди видели радиоволны вместо обычного света?"]

Выполни задачу строго по шагам:

ШАГ 1: Концепт и текст (160–180 слов) ВАЖНО СДЕЛАТЬ ИМЕННО СТОЛЬКО СЛОВ
- Придумай логические последствия этого «Что если?». Как изменился бы быт, архитектура, внешность людей?
- Напиши текст для озвучки: это должен быть философский, эпичный монолог. Начни с вопроса «Что если...» и закончи сильной финальной фразой.
- Язык: РУССКИЙ.

ШАГ 2: Константы мира (на английском)
Создай несколько переменных, неважно сколько (например, [Visual Anomaly], [Modified Human], [World Atmosphere]).
- Опиши их максимально детально: как именно выглядит это изменение. Это фундамент для консистентности.
- Используй художественные стили: (masterpiece, best quality, anime style, cinematic lighting). 
- Язык: АНГЛИЙСКИЙ.

ШАГ 3: Раскадровка (12-18 сцен)
1. text_scene: Одно полное предложение из текста Шага 1. Минимальное количество слов - 7.
2. image_prompt: Промпт для Z Image Turbo. Описание композиции, освещения и объекта с использованием [переменных]. Фокус на визуализации «иного» мира.
3. video_prompt: Промпт для Hunyuan 1.5. Описывай движение аномалий: пульсация волн, мерцание воздуха, замедленная съемка, частицы.

Ответь СТРОГО в формате JSON без markdown разметки.
